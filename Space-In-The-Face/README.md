# Space In The Face 
Space In The Face is een leuk klein arcade-stijl shoot-em-up spel dat ik heb gemaakt met behulp van het HTML canvas element. Het doel van het spel is eenvoudig: veeg de aliens uit die van boven komen. Je twee schepen zijn uitgerust met de beste militaire ruimtewapens die er zijn. Maar er is een addertje onder het gras. Om voldoende kracht te leveren om de projectielen af te vuren, moet je schip voorwaarts bewegen om te kunnen schieten...


## Aan de slag

Als je wilt spelen met de code op je eigen lokale machine, is het heel eenvoudig om de ontwikkelomgeving in te stellen.

### Vereisten

Al het werk wordt gedaan in pure HTML en JavaScript. Er worden geen bibliotheken gebruikt in het project.

### Ontwikkelen

Het geheel van het spel bevindt zich in het [javascript.js](javascript.js) bestand, met de basis canvas setup en styling in de [index.html](index.html). Alle spelresources bevinden zich in de [resources map](resources).

### Lokaal testen

Open eenvoudigweg index.html in een browser die het HTML canvas element ondersteunt, en het spel zou moeten beginnen te draaien.
